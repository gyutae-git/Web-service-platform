## 1. 문서 개요

본 문서는 lg.kimgyutae.com 프로젝트의 **프론트엔드 및 백엔드/관리자 기능을 상세히 명세**한다. 각 기능은 고유 ID, 기능명, 상세 설명, 기술적 구현 방법, 중요도를 포함하여 체계적으로 기술한다.

### 1.1 중요도 등급 정의

| 등급 | 설명 |
|------|------|
| **🔴 상 (Critical)** | 프로젝트 핵심 기능. 미구현 시 서비스 운영 불가. |
| **🟡 중 (Major)** | 사용자 경험 향상에 중요한 기능. 미구현 시 서비스 품질 저하. |
| **🟢 하 (Minor)** | 부가적 기능. 미구현 시 서비스 운영에 지장 없음. |

## 2. 프론트엔드 상세 기능

### 2.1 글로벌 헤더 (메가 메뉴)

> 모든 페이지의 최상단에 고정되는 사이트 전역 네비게이션 컴포넌트. LG Careers 스타일의 메가 메뉴를 커스텀 HTML/CSS로 구현.

| 기능 ID | 기능명 | 상세 설명 | 구현 기술 | 중요도 |
|---------|--------|----------|----------|--------|
| **FE-HDR-001** | 메가 메뉴 래퍼 | `.lg-mega-wrapper` 컨테이너. 기본 높이 65px, 100vw 너비. `position: relative`로 설정. 배경 투명. | 커스텀 CSS (`height: 65px`, `transition: height 0.4s ease`) | 🔴 상 |
| **FE-HDR-002** | 메가 메뉴 호버 확장 | 사용자가 `.lg-mega-container` 위에 마우스를 올리면 래퍼 높이가 320px로 확장되고, 배경이 흰색(#fff)으로 변경됨. CSS `:has()` 선택자 사용. | `lg-mega-wrapper:has(.lg-mega-container:hover) { height: 320px; background: #ffffff; }` | 🔴 상 |
| **FE-HDR-003** | 브랜드 로고 영역 | `.lg-brand-area` 내 LG 로고 이미지(45px 높이) + "LG" 텍스트(35px, 700 weight). 좌측 절대 위치(`position: absolute; left: 0`). 클릭 시 홈(/)으로 이동. | `<img class="lg-main-logo" src=".../LG_logo_red.png.webp">` + `<span class="lg-site-title">LG</span>` | 🔴 상 |
| **FE-HDR-004** | 메뉴 섹션 레이아웃 | `.lg-mega-container` 내 5개 `.lg-section`이 Flexbox 가로 배치. 각 섹션 `flex: 0 1 180px`. `padding-left: 180px`으로 로고 영역 확보. | Flexbox 레이아웃, `justify-content: center` | 🔴 상 |
| **FE-HDR-005** | 대분류 메뉴 제목 | `.lg-parent-title` — 18px, 800 weight, 검정. 하단 투명 보더(2px). 해당 섹션 호버 시 검정 보더 활성화. | `.lg-section:hover .lg-parent-title { border-bottom: 2px solid #000; }` | 🟡 중 |
| **FE-HDR-006** | 서브메뉴 리스트 | `.lg-child-list` — 기본 상태에서 `opacity: 0; visibility: hidden`. 메가 컨테이너 호버 시 `opacity: 1; visibility: visible`로 fade-in. | CSS `transition: opacity 0.3s` | 🔴 상 |
| **FE-HDR-007** | 서브메뉴 링크 스타일 | 15px, 500 weight, 색상 #333. 호버 시 LG 레드(#A50034)로 변경. `white-space: nowrap`. 20px 하단 마진. | `.lg-child-list li a:hover { color: #A50034; }` | 🟡 중 |
| **FE-HDR-008** | 스티키 헤더 | Blocksy 테마의 스티키 헤더 기능 활용. `data-sticky="shrink"` 속성 적용. 스크롤 시 헤더가 축소되며 상단 고정. | Blocksy Companion 스티키 헤더 JS (`blocksy_sticky_header`) | 🔴 상 |
| **FE-HDR-009** | 홈페이지 투명 헤더 | page-id-838(홈페이지)에서만 `.ct-header`, `.ct-sticky-container` 등의 배경을 투명으로 설정. `position: absolute`로 영상 위에 오버레이. | `.page-id-838 .ct-header { background: transparent; position: absolute; }` | 🟡 중 |
| **FE-HDR-010** | 반응형 메뉴 (1200px 이하) | 1200px 이하에서 `padding-left: 120px`, 섹션 `flex: 0 1 150px`, 사이트 제목 28px로 축소. | `@media (max-width: 1200px) { ... }` | 🟡 중 |
